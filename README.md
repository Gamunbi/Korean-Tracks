- 각 선로들의 의도한 주요 용도
  - 저속선:화물기차, 증기기관차
  - 일반선:일반 여객기차, 경춘선
  - 준고속선:KTX-이음 등 준고속철도
  - 고속선:SRT, KTX 등 고속철도 
  - 도시철도/지하철:대부분의 도시철도/지하철
